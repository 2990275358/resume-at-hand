# 随手简历

## 一、功能介绍

随手简历旨在帮助我更轻松的编写简历，使用 word 套用简历模版是一个比较麻烦的事情，稍不注意样式就改变了。目前也有一些免费开源的简历编辑工具，但是于我而言都或多或少有不足，最大的不足就是简历模版较少，编辑方式固定，还有一些模版较多的产品，但是使用受限，需要付费。

鉴于我目前正在学习的状态，所以决定自己实现一个简历编辑网页，希望整理和复习自己的前端知识，主要功能仿照 **全民简历** 进行实现，只实现简历模版和简历编辑的功能。

## 二、具体细节

### 1、简历模版

简历模版是由我根据网络模版的样式然后使用 **css** 实现的模版，网站会内置一套简历数据，所有模版通用的数据格式

### 2、涵盖功能

#### 2.1 编辑简历

和全民简历一致的编辑方式，不过可以随意使用想要的模版(我已经copy后的才行)

#### 2.2 编辑输入项

简历主要就是几个模块组成，模块里又是输入的子模块，我允许用户自定义模块组成，实现一个简单的低代码编辑方式

#### 2.3 数据持久化

不用登录也能随意创建简历，这时是存在本地 **indexedDB** 的，想要不同浏览器同步就需要登录后进行网络同步

#### 2.4 导出

导出主要会使用两种方式，一种是浏览器默认的导出方法，另一个种是Server利用Puppeteer将网页转成pdf

#### 2.5 其余细节

- 简历能够控制主题色和字体，还能够控制模块边距、行边距、页面边距

### 3、结语

其实项目做到中期时AI工具已经很成熟了，所以项目慢慢变成了纯为了面试准备的项目，不过项目中使用的技巧也是不错的经验，也会接入DeepSeek来AI润色简历，当然肯定是哪个大模型最便宜用哪个。

遗憾的是我没有UI设计能力，导致项目UI很丑，好在它能够正常使用，并且可以无限创建简历，每份职业都来一份。
